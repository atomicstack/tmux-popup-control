# Demos

Animated recordings of tmux-popup-control driving a real `tmux
display-popup`, captured with asciinema inside a Linux container and rendered
to looping SVG. Each demo is 160×45 and loops continuously.

## Menu overview

Open the root menu with `prefix + Space` and step into each submenu in turn —
extract, keybinding, command, pane, window, plugins, resurrect, session.

<img src="https://raw.githubusercontent.com/atomicstack/tmux-popup-control/demo-svg/overview.svg?v=47bcdac" width="100%" alt="root menu overview — animated demo">

## Command menu

`prefix + :` — type a short prefix, tab-complete the option name from the
built-in option catalog, give it a colour value, and run it. The active pane
border recolours on the spot.

<img src="https://raw.githubusercontent.com/atomicstack/tmux-popup-control/demo-svg/command-completion.svg?v=47bcdac" width="100%" alt="command menu with catalog completion — animated demo">

## Session tree

`prefix + s` — browse sessions, windows, and panes as a hierarchical tree.
The preview panel follows the cursor, showing live pane content.

<img src="https://raw.githubusercontent.com/atomicstack/tmux-popup-control/demo-svg/session-tree.svg?v=47bcdac" width="100%" alt="session tree with live preview — animated demo">

## Pane switcher

`prefix + f` — fuzzy-filter panes as you type; an async capture-pane preview
panel on the right updates to show the highlighted pane.

<img src="https://raw.githubusercontent.com/atomicstack/tmux-popup-control/demo-svg/pane-switch.svg?v=47bcdac" width="100%" alt="pane switcher with live preview — animated demo">

## Save and resurrect

`prefix + C-s` / `prefix + C-r` — snapshot a busy server (five sessions, ten
windows each) with an animated progress bar, kill the whole tmux server, then
restore everything from the snapshot — pane contents included.

<img src="https://raw.githubusercontent.com/atomicstack/tmux-popup-control/demo-svg/resurrect.svg?v=47bcdac" width="100%" alt="save and resurrect with progress bar — animated demo">

## Extract

`prefix + Tab` — extrakto-style token grabber: capture the pane, cycle the
token mode to `url` with `ctrl-f`, fuzzy-filter, and insert the selection at
the prompt with enter.

<img src="https://raw.githubusercontent.com/atomicstack/tmux-popup-control/demo-svg/extract.svg?v=47bcdac" width="100%" alt="extract token grabber in url mode — animated demo">

---

The SVGs live on the [`demo-svg`](https://github.com/atomicstack/tmux-popup-control/tree/demo-svg)
orphan branch so the main branch's history stays lean; they are regenerated by
the demo harness and force-pushed on re-recording.
